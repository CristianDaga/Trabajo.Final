### Objetivo General
El objetivo de esta acta es establecer un entendimiento común entre los integrantes del grupo para conformar un ambiente de trabajo productivo, colaborativo y armónico. A través de esta, se busca garantizar el cumplimiento de los lineamientos académicos y metas establecidas para el proyecto semestral en la materia de Algoritmos y Programación. Es fundamental que todos los miembros del grupo compartan una visión clara y coherente sobre los objetivos a alcanzar, las expectativas individuales y grupales, y la forma en que se gestionará el proyecto a lo largo del semestre.

### Expectativas del Grupo

- **Compromiso:** Todos los integrantes se comprometen a cumplir con las tareas asignadas, así como a participar activamente en todas las etapas del proyecto. Se espera que cada miembro tome la iniciativa cuando sea necesario y contribuya de manera significativa al avance del proyecto. El compromiso también implica una disposición para aprender, resolver problemas y adaptarse a las necesidades del equipo, incluso cuando surjan desafíos imprevistos.

- **Comunicación:** Se mantendrá una comunicación constante, clara y abierta entre todos los miembros del grupo para asegurar que todos estén alineados con el progreso y las decisiones del proyecto. Las vías de comunicación incluirán reuniones periódicas, mensajería instantánea y correos electrónicos. Es esencial que cualquier cambio o actualización sea compartido de inmediato para evitar malentendidos o retrasos. Además, se establecerán protocolos para la toma de decisiones y la distribución de información crítica, asegurando que todos los miembros del equipo estén informados y puedan contribuir de manera efectiva.

- **Respeto:** Se promoverá un ambiente de respeto mutuo dentro del equipo, valorando las opiniones, ideas y aportes de todos los integrantes, independientemente de su experiencia o rol en el proyecto. Se espera que las diferencias de opinión se manejen de manera constructiva, fomentando un diálogo abierto y respetuoso. El respeto también se manifestará en el reconocimiento del trabajo y esfuerzo de cada miembro, así como en la disposición para ofrecer y recibir retroalimentación de manera positiva y orientada a la mejora continua.

## Acta de Colaboración

### Colaboración y Apoyo Mutuo
Todos los miembros del grupo se comprometen a colaborar de manera constructiva y apoyarse mutuamente en la resolución de problemas. Se fomentará un ambiente de respeto, donde se valorarán las opiniones de todos y las diferencias se manejarán de manera cordial. Se compartirá el conocimiento y habilidades para mejorar el trabajo del equipo y se buscarán soluciones en conjunto para cualquier desafío que surja.

### División de Tareas
Las tareas se distribuirán equitativamente entre los miembros del grupo, considerando las fortalezas y debilidades de cada uno para optimizar el rendimiento del equipo. Se asignarán responsabilidades que permitan a cada miembro contribuir de manera efectiva, y se ajustarán si es necesario para garantizar que todos participen de forma justa.

### Reuniones
El grupo acordará un horario conveniente para todos y se comprometerá a asistir puntualmente a las reuniones. La comunicación será constante y abierta, utilizando tanto reuniones en línea como en persona, según se acuerde. Las reuniones servirán para revisar el progreso, resolver problemas y tomar decisiones en conjunto.

### Resolución de Conflictos
En caso de conflictos o desacuerdos, se seguirá el siguiente procedimiento:
1. Discutir el problema abiertamente en una reunión para intentar llegar a una solución consensuada.
2. Si el conflicto persiste, se buscará la mediación de un tutor o profesor para resolver el problema de manera justa.

Este proceso garantizará que cualquier desacuerdo se maneje de manera efectiva y equitativa, buscando siempre la mejor solución para el grupo y el proyecto.
